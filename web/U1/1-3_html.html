Was ist HTML?

HTML steht für Hypertext Markup Language. HTML ist eine Sprache die von einer Maschine gelesen werden kann und zur Gliederung und Formatierung von Texten und anderen Daten dient.

Wie ist eine HTML-Tabelle aufgebaut?

Eine HTML-Tabelle ist in table rows = tr, tabel header = th und tabel data = td aufgeteilt

                <table>
                    <tr>
                        <th>header</th>
                    </tr>
                    <tr>
                        <td>data</td>
                    </tr>
                </table>
header
data

Welche Konventionen sollte man bei Pfaden und Dateinamen beachten?

Wenn möglich relative Dateipfade nutzen und Dateinamen klein schreiben.

Wie baut man in HTML ein Menü?

Mit einer unordered list und links als list-item.

            <h2>Navigation Menu</h2>

            <ul>
                <li><a href="#home">Home</a></li>
                <li><a href="#news">News</a></li>
                <li><a href="#contact">Contact</a></li>
                <li><a href="#about">About</a></li>
            </ul>

            https://www.w3schools.com/html/tryit.asp?filename=tryhtml_lists_menu -->

Welche Attribute sollte man bei Bildern wie verwenden?

            <img src="bild.jpg" alt="Bild" style="width:500px;height:600px;">

            Alt als Ersatz falls Bild nicht geladen werden kann.
            Außerdem height und width immer angeben, allerdings mit style=".." anstatt height=".." und width=".."
