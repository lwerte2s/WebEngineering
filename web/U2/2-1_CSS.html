2.1. CSS Selektoren und CSS Farben

                Berechnen Sie die Spezifität folgender CSS-Selektoren:
                
                div div div:focus .inner
                h1 + div.main
                div a:link[href*='h-brs']
                nav > a:hover::before
                ul#primary-nav li.active
            
                div div div:focus .inner = 23
                h1 + div.main = 12
                div a:link[href*='h-brs'] = 22
                nav > a:hover::before = 13
                ul#primary-nav li.active = 112
            
                Rechnen Sie folgende RGB-Werte in HSL-Werte um:
                
                #ffffff
                #000
                #ab0978
                rgb(127,255,33)
                rgba(255,127,33,0.8)
            
                #ffffff = (0°, 0%, 100%)
                #000 = (0°, 0%, 0%)
                #ab0978 = rbg(171,9,120) = (319°,90%,35.3%)
                rgb(127,255,33) = (95°, 100% ,56.5%)
                rgba(255,127,33,0.8) = (25°, 100%, 56.5%, 0.8)
